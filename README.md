##  &#128214; JustBook
국내 온라인 서점들이 여러 광고와 책 이외에 다른 물건들을 판매하는것을 보았습니다. 이를 계기로 책 구매에 집중할 수 있는 온라인 서점 앱을 기획, 개발하게 되었습니다.

**해당 어플리케이션에 대한 자세한 내용은 Wiki에 정리했습니다.**

## &#128106; 기능 
- 회원가입
- 로그인
- 회원 정보 수정
- 회원 탈퇴
- 카테고리별 책 구매순, 좋아요 순 조회
- 장바구니
- 상품 결제
- 상품 검색
